# 出發前確認事項

## 證件與簽證
- [ ] 護照（效期需超過入境日 6 個月以上）
- [ ] K-ETA 申請（韓國電子旅行許可，出發前 72 小時完成）
- [ ] 電子機票列印或截圖

## 交通與通訊
- [ ] T-money 交通卡（機場購買或事先準備）
- [ ] 韓國上網 eSIM 或 WiFi 分享器
- [ ] 下載 Naver Map APP（韓國地圖首選）
- [ ] 下載 KakaoMap APP（備用導航）

## 生活用品
- [ ] 韓國轉接頭（Type C/F 圓腳兩孔）
- [ ] 行動電源
- [ ] 雨具（三月仍有機會下雨）
- [ ] 外套（三月釜山早晚溫差大，約 5~15°C）

## 金錢
- [ ] 韓幣兌換（台幣→韓幣，建議在台先換）
- [ ] 信用卡（韓國刷卡率高，Visa/Mastercard 為主）
- [ ] 跨國提款卡（備用）
