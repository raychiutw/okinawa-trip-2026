# 出發前確認清單

## 板橋景點預約
- [ ] 林本源園邸（Day 1）：需提前上網購票，旺季週末常售罄，建議至少提前一週預約
- [ ] 確認林本源園邸當日導覽時段與費用（全票100元）
- [ ] 板橋435藝文特區（Day 1）：免費入園，確認特展是否需另購票

## 花季景點確認 {sand}
- [ ] 土城桐花公園（Day 10）：油桐花通常四月上中旬盛開，出發前查詢當年花況
- [ ] 幸福水漾公園（Day 6）：花海季節約二至四月，提前確認四月初花況
- [ ] 建議下載新北市政府 APP 或查詢新北旅遊官網掌握花況即時資訊

## 夜市與市場開放日
- [ ] 湳雅觀光夜市（Day 3）：每週四、六、日有較多攤販，建議確認當日營業情況
- [ ] 樂華夜市（Day 9）：每天開放，但週末攤販最齊全
- [ ] 樹林興仁花園夜市（Day 11）：週三、五、六、日營業，4/11（六）正常開放
- [ ] 新莊廟街（Day 5）：確認廟宇參觀開放時間，部分廟宇早晚不定時關閉

## 隨身備品 {sand}
- [ ] 悠遊卡或電子票證（部分停車場可用）
- [ ] 防曬乳SPF50+、帽子、太陽眼鏡（四月新北日照強）
- [ ] 輕便雨傘或雨衣（四月梅雨季前期，午後雷陣雨常見）
- [ ] 行動電源＋充電線
- [ ] 運動鞋（烘爐地、桐花公園、大安圳步道等需健走）

## 博物館與古蹟確認
- [ ] 山佳車站（Day 12）：市定古蹟，確認鐵道文化園區開放時間
- [ ] 空軍三重一村（Day 6）：確認開放時段，部分日期可能有特展導覽
- [ ] 十三行博物館（Day 15）：全票80元，每月第一個週日博物館日免費入場
- [ ] 紅毛城（Day 13）：國定古蹟，確認開放時間與票價
